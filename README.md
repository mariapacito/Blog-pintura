Blog Pintura - CAPS
Projeto de back-end solicitado para a atividade.

Como rodar o projeto:
npm install (Para baixar as dependências)

npm run backend (Para ligar o servidor Express na porta 3001)

npm run dev (Para ligar o site em Next.js na porta 3000)

Estrutura:
O arquivo do servidor Express está em: backend/server.js

O Schema do Prisma está em: backend/prisma/schema.prisma
